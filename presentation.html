AWS Lambda SQS Dashboard

Comprehensive Management Platform for AWS Lambda Functions and SQS Queues

Project Overview

What is it?

A modern, full-stack web application for managing AWS Lambda functions, SQS queues, and Alert Logic collectors across multiple AWS profiles and regions.

Key Capabilities

  • Multi-Account Management: Switch between AWS profiles dynamically
  • Real-time Monitoring: View Lambda functions, SQS queues, and collector status
  • Deployment Management: Update Lambda code from S3 zip files
  • Queue Operations: Poll messages, toggle triggers, manage visibility timeouts
  • CloudWatch Integration: View logs directly in the dashboard
  • Collector Administration: Manage Alert Logic collectors and inactive users

Performance Optimizations

AWS SDK v3 Optimizations

  • Client Caching: Reuse AWS SDK clients for connection pooling
  • Intelligent Polling: Check queue counts before polling SQS
  • Timeouts + Retries: 3s connection, 30s socket, limited attempts

Frontend Optimizations

  • Memoization: useMemo/useCallback for heavy renders
  • Debounced Search: Reduce unnecessary API calls
  • Pagination: Limit data displayed per page (50 items)

Testing Strategy

Test Layers

  • Unit: Data formatters, AWS helpers, table filters, pagination logic
  • Integration: API routes with mocked AWS SDK v3 clients
  • E2E: User flows across dashboard, SQS tools, and zip update

Suggested E2E Coverage

  • Load dashboard and list Lambda functions
  • Change profile and region and refresh data
  • Open Lambda details flyout and load logs
  • Toggle SQS triggers and verify status updates
  • Zip update: fetch zip files, select Lambda, deploy
  • Collector admin: search, update config, view results

Tools: Playwright or Cypress for E2E, Jest or Vitest for unit/integration.
